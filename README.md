# ☁️ Monitor de Clima em Tempo Real com Python & Streamlit

###  Descrição do Projeto
Desenvolvimento de uma aplicação web responsiva voltada para a consulta de dados meteorológicos em tempo real. O sistema consome a API global da **OpenWeatherMap** para buscar informações climáticas detalhadas de qualquer cidade informada pelo usuário, realizando o tratamento e a exibição dos dados de forma limpa e dinâmica.

###  Tecnologias e Ferramentas Utilizadas
* **Linguagem:** Python
* **Interface Web:** Streamlit
* **Consumo de API:** Biblioteca `requests`
* **Formatação de Dados:** Tratamento de strings, estruturas condicionais e dicionários.

###  Principais Desafios Superados
* **Consumo e Tratamento de Dados de API:** Integração com serviços externos, manipulação de responses em formato JSON e conversão matemática de unidades (Kelvin para Celsius) em tempo real.
* **Resolução de Conflitos de Ambiente:** Ajuste de variáveis de ambiente e configurações de sistema operacional para garantir a estabilidade e portabilidade do projeto.
